# official-macaujc-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态网页的形式呈现，内容涵盖各类笔记、文档或参考资料，供有需要的用户查阅和使用。

仓库不针对任何特定域名或网站，仅作为页面内容的集中存放与版本管理平台。

## 目录说明

```
├── index.html          # 仓库主页（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 页面所需的资源文件（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录用于存放所有独立的 HTML 页面文件，每个文件可视为一个独立的内容单元。
- `assets/` 目录用于存放页面共享或专用的静态资源，便于统一管理。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接在浏览器中打开浏览。
- 页面之间无强制依赖关系，可根据需要单独引用或归档。
- 归档内容可能不定期更新，更新记录可通过 Git 提交历史查看。

## 维护说明

- 本仓库由维护者定期更新，添加新的页面或修正已有内容。
- 欢迎通过 Issues 提交问题反馈或改进建议。
- 不建议直接修改仓库文件，如需贡献，请先联系维护者。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器打开 `pages/` 目录下的任意 HTML 文件。
3. 也可将仓库部署到任意静态托管服务，实现线上访问。
